# VEX Reports Repository

This repository contains vulnerability exchange (VEX) reports for Evertrust products, automatically built and published as GitHub releases.

### Release Format

Releases are named: `vex-reports-YYYY-MM-DD`

Each release includes:

- CSV files for each product (e.g., `oci_horizon.csv`)
- Trivy ignore files for each product (e.g., `oci_horizon.trivyignore.yaml`)
- Build metadata (`build_info.txt`)
